A simple IP Geolocation tool.

![Screenshot](https://raw.githubusercontent.com/haithamaouati/IPWhoIs/main/screenshot.PNG?raw=true "Optional Title")

Requirements
----
* [Python 3.9.4](https://www.python.org)

Modules
----
    pip3 install socket
    pip3 install colorama
    pip3 install urllib2
    
Installation
----
    git clone https://github.com/haithamaouati/IPWhoIs.git
    cd IPWhoIs
    chmod+x IPWhoIs.py
    
Usage
----
    python3 IPWhoIs.py -i <ip>
    
Help
----
    python3 IPWhoIs.py -h
    
###### Options
`-h, --help`
show this help message and exit.

`-i, --ip`
IP used for the query (e.g. 8.8.4.4)

Platform
----
* Windows
* Linux
* macOS
